Hướng dẫn Tạo 1 project trên github
	New -> Repository name 
	Public || private 
	Đẩy file lên github:
		git status -> kiểm tra trạng thái file (sẽ đổi màu file vừa thay đổi)
		git add . -> thêm toàn bộ file -> gõ lại git status
		git commit -m "" -> ghi chú lại nội dung file vừa thêm là gì ?
		git push origin main -> đẩy lên github
Hướng dẫn tạo project local và đẩy lên github 
	Tạo 1 github reponsitory
	git init -> tạo 1 empty git repo
	git status -> git add . -> git commit -m ""
